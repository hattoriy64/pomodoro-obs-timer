# Pomodoro Timer for OBS / YouTube Streaming

YouTube配信・OBSブラウザソース向けのポモドーロタイマーです。  
背景が完全透過のため、映像ソースの上にオーバーレイ表示できます。

![フェーズ: 作業中は青系、休憩中は緑系にアクセントカラーが変化します]

---

## 機能

- **作業・休憩時間** を個別に設定（1〜120分）
- **作業中 → 休憩中 → 作業中** の自動サイクル
- **完了セット数** のカウント・リセット
- 開始 / 一時停止 / リセット ボタン
- OBSブラウザソース向け **1280×200px 横長レイアウト**
- 背景 **完全透過**（`background: transparent`）

## セットアップ

### 前提条件

- [Node.js](https://nodejs.org/) 18以上
- npm

### インストール

```bash
npm install
```

### ビルド

TypeScript → JavaScript にコンパイルします。

```bash
npm run build
```

### 開発（ウォッチモード）

ファイル変更を監視して自動でリコンパイルします。

```bash
npm run watch
```

## OBS ブラウザソース 設定手順

### 1. ファイルをローカルに配置

リポジトリをクローン（またはダウンロード）し、`npm run build` でビルドを完了させてください。

```bash
git clone https://github.com/<あなたのユーザー名>/pomodoro-obs-timer.git
cd pomodoro-obs-timer/pomodoro
npm install
npm run build
```

### 2. OBSにブラウザソースを追加

1. OBS Studioを開き、**ソース** パネルの `+` をクリック
2. **ブラウザ** を選択し、ソース名を入力して **OK**

### 3. ブラウザソースの設定

| 項目 | 設定値 |
|------|--------|
| **ローカルファイル** | チェックを入れる |
| **ファイルパス** | `C:\...\pomodoro\index.html`（フルパスを指定） |
| **幅** | `1280` |
| **高さ** | `200` |
| **カスタム CSS** | （空欄のまま） |
| **シーン切り替え時にブラウザをリフレッシュ** | 任意 |

> **ヒント**: ファイルパスは「参照」ボタンから `index.html` を選択すると楽です。

### 4. 配置

ソースを映像キャプチャの上層に配置し、画面下部や上部にドラッグして完成です。

## 使い方

| ボタン | 動作 |
|--------|------|
| **開始** | タイマーをスタート |
| **一時停止** | タイマーを一時停止 |
| **リセット** | 現フェーズをリセット（作業中に戻る） |
| **セットリセット** | 完了セット数を 0 に戻す |

- **作業・休憩の分数入力欄** はタイマー停止中に変更できます
- フェーズが変わると自動で次のカウントダウンが始まります
